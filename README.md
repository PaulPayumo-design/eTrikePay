# eTrikePay

Blockchain-based fare payment system for tricycle and e-trike drivers using Stellar.

---

## Problem

Drivers in the Philippines lose income due to fake payment screenshots, delayed transfers, and lack of exact cash.

---

## Solution

Passengers scan a QR code and pay fares instantly using Stellar USDC/XLM with transparent on-chain payment verification.

---

## Timeline

- Day 1: Smart contract
- Day 2: Frontend UI

## Stellar Features Used

- XLM transfers
- Soroban smart contracts

---

## Vision and Purpose

Enable secure and instant transportation payments for small public transport operators in Southeast Asia.
